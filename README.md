## Getting Started
**Install with [npm](https://npmjs.org/)**
```bash
npm install avdan
```
***or with [yarn](https://yarnpkg.com/)***
```bash
yarn add avdan
```

## Development
**Build the library**

```
npm run rollup
```

**Publish the library**

```
npm publish
```

**Run tests**

```
npm run test
```

**Run storybook locally**

```
npm run storybook
```

**Build storybook**

```
npm run build-storybook
```
